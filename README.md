Together - Crowdfunding MVP

This workspace contains a minimal MERN scaffold: `backend/` and `frontend/`.

Backend
- cd backend
- copy `.env.example` to `.env` and adjust values (MongoDB URI and JWT secret)
- npm install
- npm run dev

Frontend
- cd frontend
- npm install
- npm run dev

By default the frontend assumes the backend API is at `http://localhost:5000/api`.
